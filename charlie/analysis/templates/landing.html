About us

Charlie's primary function entails manipulating data and then representing it in many different forms. It also allows the user to edit the dataset while in session and provides him/her with the revised version for future use. This not only makes it easier for the user to keep working on an updated dataset, but also makes the process a lot more effortless and enjoyable. Although we came across Charlie as a IB Group Project, it soon grew to be something bigger for all of us. We hope that you will find it helpful and that you enjoy using it as much as we enjoyed making it.
